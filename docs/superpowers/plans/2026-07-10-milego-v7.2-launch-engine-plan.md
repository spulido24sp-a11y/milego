# Implementation Plan: Sprint 2 — LIAM Launch Engine

Este plan técnico detalla la arquitectura, contratos de datos, modelo interno y estructura de carpetas para el **LIAM Launch Engine**. Este motor procesa y transforma cualquier producto en un lanzamiento comercial completo de forma independiente del proveedor de origen (Dropi, Triidy, CSV o ingreso manual).

## 🧠 Los Tres Niveles de Inteligencia (Triple DNA Model)

Cada lanzamiento en el sistema se analiza en tres dimensiones para alimentar las decisiones de LIAM:

### 1. Product DNA
*   Propuesta de valor técnica, SKU, fotos, costo de proveedor, stock y dimensiones físicas.

### 2. Market DNA
*   Precios de competidores en Colombia, volumen de demanda estimado, anuncios activos en el sector, palabras clave SEO de alto tráfico local y estacionalidad.

### 3. Customer DNA
*   Frecuencia de recompra, dolores y deseos emocionales, objeciones típicas ante el producto, disparadores de compra por impulso y perfil sociodemográfico idóneo.

## 📝 Estructura de Business Memory (Memoria del Negocio)

La memoria comercial almacena de forma inmutable los resultados de cada combinación para alimentar las decisiones de LIAM:

```json
{
  "launch_id": "integer",
  "product_id": "integer",
  "combination": {
    "hook_id": "string",
    "color_palette": "string",
    "offer_type": "combo_2x | combo_3x | discount"
  },
  "metrics": {
    "ctr": 5.8,
    "roas": 4.3,
    "conversion_rate": 3.8
  },
  "outcome": "success | failure | neutral"
}
```
